Examen Git y Github

Alumn@: Germán Enrique Faciano

  1. Que es un sistema de control de versiones?

    Es una aplicación que permite llevar seguimiento de nuestro código en sus distintos estadíos, y en caso de necesitar podemos hacer rollback de los cambios.

  2. Cual es la diferencia entre git y github?

    Git es la aplicación (interfaz) que nos brinda los comandos para gestionar las versiones de nuestro código. GitHub es la plataforma de versionado que se gestiona con aplicaciones como Git.

  3. Cómo creo una rama en git?

    utilizando el comando git checkout -b "nombre_rama"

  4. Cómo clono un repositorio?

    utilizando el comando git clone "url_repository"
