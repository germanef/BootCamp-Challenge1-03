  1. Que es un sistema de control de versiones?

    Un sistema de control de versiones es un software que lleva un seguimiento y un control de todos los cambios en un archivo y/o proyecto.

  2. Cual es la diferencia entre git y github?

    Git es un sistema de control de versiones, y GitHub es un servidor online de repositorios de git.

  3. Cómo creo una rama en git?

    Una nueva rama se crea con el comando git checkout -b nombre-de-la-nueva-rama. Este comando no solo crea la rama si no que con el parametro -b salta a la nueva rama creada.
